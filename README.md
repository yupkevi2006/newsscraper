# 📰 NewsScraper

A Python-based tool that automatically fetches and extracts the latest news articles using RSS feeds.

## 🚀 Features
- Fetches real-time news from RSS feeds
- Extracts full article content
- Displays title, summary, and article link
- Handles errors gracefully

## 🛠️ Tech Stack
- Python
- Newspaper3k
- Feedparser

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python main.py

## 📌 Example Output
- Title: Sample News Title
- Summary: Extracted article text...
- Link: https://example.com
